# Dungeon Panda

Background music for tabletop gaming, using your Apple Music collection. There are many other applications like this, but Dungeon Panda does its best to be very simple and reliable, with an intentionally threadbare feature set.

## Compilation

**IMPORTANT:** Once dependencies are installed, you must open the `.xcworkspace` file in XCode, not the `.xcodeproj` file; for dependencies:

* If necessary install CocoaPods via `gem install cocoapods` / see https://cocoapods.org if there are problems
* Install dependencies with `pod install`
* Open the `Dungeon Panda.xcworkspace` file in XCode.
